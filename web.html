MyDictionary

MyDictionary is a text-based social media platform. It allows users to create and share definitions, acronyms, and quotes created by themselves and others. I created MyDictionary in my CMSC 508 Databases class at VCU using PHP, SQL, HTML, and CSS.

Qdoba Job Application

A website for a local QDoba that allows users to fill out and submit an online job application. When a new application is submitted it is saved to Google Sheets, an email notification is sent to the manager, and the manager can then view the application. I created this website using HTML, CSS, Javascript, and Google Scripts
